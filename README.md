# BrandBasket – MERN E-commerce Website

BrandBasket is a MERN stack based e-commerce web application for shopping different brands of clothing and accessories for men, women, and kids. The website includes product listing, category filtering, authentication, cart functionality, and separate admin and user panels.

---

## Features

* User authentication and login
* Product listing and category filtering
* Add to cart functionality
* Admin panel and user panel
* Brand and product management
* Payment integration using PayPal
* Responsive UI design

---

## Categories

* Men Clothing
* Women Clothing
* Kids Clothing
* Shoes
* Watches
* Accessories

## Project Setup

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```
